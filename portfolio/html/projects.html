My Projects

Daycare Management System

Description: The Daycare Management System is a desktop-based application developed using C# (.NET Framework) and MySQL to simplify the daily operations of a daycare center. The system helps manage parents, children, and caretakers in a centralized platform. Parents can create accounts, add their children's information, upload photos, and search for available caretakers. Caretakers can manage their profiles, update availability, and accept requests. The system also allows administrators to monitor records and maintain the overall daycare database. This project reduces manual paperwork, improves communication between parents and caretakers, and provides a more organized and efficient daycare management process.

Skills Used: C# (.NET Framework, Windows Forms), MySQL Database, Object-Oriented Programming

3D Shooting Game

Description: Spy Detective - Shoot Stones is an interactive shooting game developed using OpenGL and GLUT. The game features a spy character who moves on a 2D gameplay plane while interacting with a 3D-rendered environment. My main contribution to this project was designing and implementing the Spy Gunman shooting system. This includes creating the spy character model, developing a mouse-based aiming mechanism, and implementing accurate bullet firing from the gun muzzle. The bullets move in real time based on direction vectors, providing a smooth and realistic shooting experience. The project demonstrates key computer graphics concepts such as geometric transformations, rotation, coordinate conversion, animation, and real-time interaction.

Skills Used: C++, OpenGL, GLUT, Game Logic

Travel and Ticket Management System

Description: The Travel and Ticket Management System is a desktop-based software developed using C# Windows Forms and SQL Server. It is designed to automate the flight ticket booking process for travel agencies and customers. The system supports three types of users: Customer, Agent, and Admin. Customers can register, search flights, book tickets, and view their booking history. Agents can manage multiple bookings, update passenger details, and handle group reservations. Admins have full control over the system, including managing users, monitoring bookings, and handling reviews or complaints. This system replaces traditional manual ticketing methods with a secure, centralized, and automated solution, improving accuracy, speed, and user experience.

Skills Used: C# (Windows Forms), SQL Server, Object-Oriented Programming, Agile (Scrum) Development Model, Figma
